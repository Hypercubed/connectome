A visual guide to FANTOM5 Ligand-Receptor interactions

* Entity selection panel * Use the toggle button on the bottom right to show and hide the entity selection and options panel. * Switch between lists of nodes, genes, and ligand receptor pairs using the tabs. * Turn on and off nodes, genes, and ligand receptors pairs using the toggle button on the left in each list. * Use the arrow on the upper right of the data list to filter each lists. * Switch between hive and force directed graph in the options tab. * Download the visualization or data in the options panel. * Information panel * Use the toggle button on the bottom right to show and hide the information panel. * This panel will display information about the current network as well as additional information no selected nodes. * Add or remove genes by clicking on gene names in the information box. * Visualization * Click to select a node. Click again to deselect. * Hold the Ctrl key to selected multiple nodes. Shift-Click to select a node and neighbors. * Right click on a node to use the context menu. * Use the context menu to show top neighbors.
